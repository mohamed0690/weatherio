## Introduction
Welcome to Weatherio, your go-to weather web app for up-to-date weather information! This project was created as a part of the Front-end Development track at #Route Academy, using HTML, CSS, and Vanilla JavaScript. Weatherio provides you with automatic location-based weather updates, forecasts for the next two days, and the ability to explore weather conditions in any city.
